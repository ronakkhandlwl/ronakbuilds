No special prerequisites needed! This is a static website that can be opened directly in any modern web browser.

1. Clone the repository
```bash
git clone https://github.com/ronakkhandlwl/portfolio.git
```

2. Navigate to the project directory
```bash

```

3. Open `index.html` in your browser
```bash

xdg-open index.html

# On Windows
start index.html
```

Or simply double-click the `index.html` file.

## ğŸ“ Customization

To customize this portfolio for your own use:

1. **Personal Information**: Update name, title, and description in the hero section
2. **Skills**: Modify the skill tags in the "Tech Arsenal" section
3. **Experience**: Update work history in the timeline
4. **Projects**: Replace project cards with your own projects
5. **Links**: Update social media links and email address
6. **Resume**: Replace `/ronaks_resume.pdf` with your own resume file
7. **Colors**: Modify CSS variables in the `:root` selector to change the color scheme

### Color Scheme Variables
```css
:root {
  --bg-light: #faf7f4;
  --bg-surface: #ffffff;
  --primary-dark: #2f2a25;
  --accent: #8b5e34;
  --accent-hover: #734c2a;
  --text-muted: #6c6c6c;
  --border-color: #e5e0db;
}
```
